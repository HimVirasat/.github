## About
We are an open-source community working to document, preserve, and digitize Himachali dialects by building open translation datasets between local dialects and Hindi.

## Our Goals
- Create freely available parallel datasets for Himachali dialects
- Enable research, education, and language technology
- Ensure these dialects are represented in the digital world

## How We Work
- Community-contributed sentences via simple forms
- Open datasets released under permissive licenses
- Technical contributors help clean, structure, and model the data
