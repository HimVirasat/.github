# Contributing Guide

Thank you for your interest in contributing to the Himachali Languages Initiative.

## Non-Technical Contributions
You can contribute by submitting sentences in your dialect along with Hindi translations using the provided forms. No technical knowledge is required.

## Technical Contributions
Technical contributors can help with:
- Dataset cleaning and validation
- Training and fine-tuning language models (LLMs)
- Building websites or apps for translation and language learning
- Designing gamified tools for language engagement
- Documentation and tooling

Please open an issue or pull request in the relevant repository.

## Code of Conduct
Be respectful, inclusive, and constructive. Linguistic diversity is a strength.
